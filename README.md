# OnionMaster
OpenTK RPG in C# .NET

## Installera projektet i Visual Studio

Det verkar vara något problem med att göra NuGet restore på NUnit. Om det uppstår, avinstallera NUnit och installera det på nytt.

## Editera spelvärlden

Spelvärlden beskrivs i SessionData.json.

Exempel på hur dessa filer skrivs syns i https://github.com/Ziretech/OnionMaster/tree/master/OnionMasterTest/Resources

Filernas effekt på spelvärlden beskrivs sedan i kodexemplen i https://github.com/Ziretech/OnionMaster/blob/master/OnionMasterTest/SessionDataConverterTest.cs

För att ändra SessionData.json så kan en editor som [Visual Studio Code](https://code.visualstudio.com/download) vara användbar.
